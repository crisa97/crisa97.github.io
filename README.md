# Dork Tool And More

![Dork Tool Preview](/img/Dork_Tool.png) <!-- Puedes reemplazar esta URL con una captura de pantalla de tu herramienta -->

**Dork Tool And More** es una herramienta web diseñada para facilitar la búsqueda de información en la web utilizando Google Dorks. También incluye funcionalidades de OSINT (Open Source Intelligence) para obtener información sobre dominios, subdominios, DNS, IP, y más.

---

## 📌 Características

- **Búsqueda con Google Dorks**: Utiliza una amplia variedad de dorks predefinidos para encontrar información sensible, archivos, credenciales, paneles de administración, y más.
- **OSINT**: Herramientas integradas para obtener información sobre subdominios (crt.sh), registros DNS, información de IP, y datos WHOIS.
- **Interfaz intuitiva**: Diseño responsive que se adapta a móviles y escritorio.
- **Copiar y buscar**: Copia los dorks o ábrelos directamente en Google con un solo clic.
- **Limpieza rápida**: Botón para borrar todos los resultados y reiniciar la herramienta.

---

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura de la herramienta.
- **CSS3**: Estilos y diseño responsive.
- **JavaScript**: Lógica de la herramienta y manejo de eventos.
- **LocalStorage**: Guarda el estado de las búsquedas de OSINT.

## 📥 Instalación

No necesitas instalar nada para usar esta herramienta. Simplemente abre el archivo `index.html` en tu navegador o visita la página alojada en GitHub Pages.

### Requisitos

- Un navegador moderno (Chrome, Firefox, Edge, etc.).

---

## 🚀 Cómo Usar

1. **Ingresar un dominio**: Escribe el dominio que deseas investigar en el campo de texto (ej. `test.com`).
2. **Filtrar resultados**: Usa el campo de filtro para buscar dorks específicos.
3. **Seleccionar una categoría**: Haz clic en las categorías para ver los dorks disponibles.
4. **Buscar o copiar**: Usa los botones "Search" para abrir el dork en Google o "Copy" para copiarlo al portapapeles.
5. **Herramientas OSINT**: Usa las herramientas integradas para obtener información sobre subdominios, DNS, IP, y WHOIS.
6. **Limpiar todo**: Usa el botón "Clear All" para reiniciar la herramienta.
